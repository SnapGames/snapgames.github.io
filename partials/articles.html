03TileMap Editor (and more...)

I've juste started th development of the Tool side. I first work on the Level editor, mainly a TileMap editor, as a start point. It would provide Tilemap editor feature (sic), TileSet autosliced, and some Layer feature, largely inspired from Tiled.

The Editor will be based on Java SWT eclipse library.

Bellow a small capabilities list, that would be ilplemented in a next release:

  1. Editors
    1. Meta-data editor
    2. Level
    3. TileSet
    4. TileMap
      1. TileSet list
      2. Layers list
      3. MapObjects hierarchical list
      4. Object meta-data template
  2. more to come ...

02Platform 2D

Platform is a 2D Game Engine attempts. based on the fabulous tutorials from the ForeignGuyMike from youtube

The SnapGames Dev Team deal was to create a 2d game platform from this work, and provide a fun tooling to develop old school like games.

The first try is for the Artifact ForeignGuyMike demo game. Totally rewritten based on the SnapGames Platform 2D engine, Artifact is .. the same :)

But build with a nice reusable in-house game engine.
